## Summary

`parakeet-cli` is a fast, fully local speech-to-text CLI for Apple Silicon macOS using NVIDIA Parakeet TDT via ONNX Runtime.

This first release includes:
- WAV file transcription with text or JSON output
- Live microphone transcription with Silero VAD
- Daemon mode for hotkey-triggered dictation
- Optional clipboard output
- Experimental CoreML support
- Homebrew install support via `lucataco/tap/parakeet-cli`

## Install

```bash
brew install lucataco/tap/parakeet-cli
```

Or build from source:

```bash
git clone https://github.com/lucataco/parakeet-cli.git
cd parakeet-cli
cargo build --release --bin parakeet
```

## Quick Start

```bash
parakeet download
parakeet devices
parakeet listen
```

Transcribe a file:

```bash
parakeet transcribe recording.wav
```

## Production Hardening Included In This Release

- pinned and checksum-verified model and VAD downloads
- safer per-user daemon socket and PID paths
- improved local file and socket handling for daemon control
- runtime panic-path cleanup for malformed input and model edge cases
- release metadata and packaging cleanup for Homebrew readiness

## Notes

- macOS on Apple Silicon is the supported target
- model files are downloaded on first use and cached locally
- daemon runtime files default to `~/Library/Application Support/parakeet/run/`
