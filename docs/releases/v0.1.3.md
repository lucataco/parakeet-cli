## Summary

`parakeet-cli` v0.1.3 is a small patch release that makes the default Parakeet model download lighter while preserving an explicit FP16 override.

## What's Changed

- changed `parakeet download` to fetch the INT8 weights by default
- added `parakeet download --fp16` as the explicit override for the larger FP16 weights
- updated the CLI help text and README to match the new default download behavior
- added CLI regression coverage for the new default and FP16 override

## Install / Upgrade

```bash
brew update
brew upgrade lucataco/tap/parakeet-cli
```

Or build from source:

```bash
git clone https://github.com/lucataco/parakeet-cli.git
cd parakeet-cli
cargo build --release --bin parakeet
```

## Quick Start

```bash
parakeet download
parakeet devices
parakeet listen
```

## Notes

- macOS on Apple Silicon is the supported target
- `parakeet download` now fetches the INT8 Parakeet TDT 0.6B v3 model by default
- use `parakeet download --fp16` if you want the larger FP16 weights instead
- the Silero VAD model is downloaded automatically on first `parakeet listen` or `parakeet serve`
