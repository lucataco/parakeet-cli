## Summary

`parakeet-cli` v0.1.1 is a small patch release that makes the project consistently v3-only and tightens a few user-facing release details.

## What's Changed

- removed the remaining benchmark fallback to the legacy v2 model directory
- added a regression test to lock the CLI default model path to `parakeet-tdt-0.6b-v3`
- corrected release messaging so `parakeet download` is the explicit Parakeet v3 setup step
- clarified that the Silero VAD model is auto-downloaded on first `parakeet listen` or `parakeet serve`

## Install / Upgrade

```bash
brew update
brew upgrade lucataco/tap/parakeet-cli
```

Or build from source:

```bash
git clone https://github.com/lucataco/parakeet-cli.git
cd parakeet-cli
cargo build --release --bin parakeet
```

## Quick Start

```bash
parakeet download
parakeet devices
parakeet listen
```

## Notes

- macOS on Apple Silicon is the supported target
- `parakeet download` fetches and caches the Parakeet TDT 0.6B v3 model locally
- the Silero VAD model is downloaded automatically on first `parakeet listen` or `parakeet serve`
