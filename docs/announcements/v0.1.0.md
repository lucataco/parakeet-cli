`parakeet-cli` v0.1.0 is out.

A fast, fully local speech-to-text CLI for Apple Silicon macOS using NVIDIA Parakeet TDT via ONNX Runtime.

Highlights:
- WAV transcription
- live microphone transcription
- daemon mode for hotkey dictation
- optional clipboard output
- Homebrew install support

Install:

```bash
brew install lucataco/tap/parakeet-cli
```

Quick start:

```bash
parakeet download
parakeet devices
parakeet listen
```
